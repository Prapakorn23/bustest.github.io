คำแนะนำในการแก้ไข

1. ตรวจสอบอุปกรณ์ GPS ของรถที่มีปัญหา

  • รถที่มี GPS = (0, 0) บ่อยครั้ง อาจมีปัญหาที่อุปกรณ์
  • ตรวจเช็คการเชื่อมต่อ GPS module

2. ตรวจสอบจุดที่มีปัญหาบ่อย (Hotspots)

  • อาจมีสิ่งกีดขวางสัญญาณ GPS (อาคารสูง, ต้นไม้หนา)
  • พิจารณาติดตั้ง GPS relay หรือปรับปรุงสัญญาณ

3. ปรับปรุงระบบบันทึกข้อมูล

  • ใช้ค่า accuracy ในการกรองข้อมูลที่ไม่น่าเชื่อถือ
  • เพิ่มการ validate ข้อมูลก่อนบันทึก (ตรวจสอบว่าพิกัดอยู่ในเขต CMU)

4. สำหรับ Data Analytics

  • ใช้ Google Maps API Snap to Roads เพื่อปรับแก้เส้นทาง
  • ใช้ Kalman Filter เพื่อ smooth GPS trajectory
  • ลบข้อมูลที่เห็นได้ชัดว่าผิดปกติ (GPS = 0, teleporting)
